# Säkerhetsrapportering

Hittar du en säkerhetsbrist i projektet eller den publika tjänsten ska du rapportera den privat.

## Kontakt

- E-post: `liam@utbildningsintyg.se`

## Inkludera gärna

1. En kort beskrivning av problemet.
2. Steg för att återskapa felet.
3. Vilken miljö som berörs, till exempel lokal utveckling, Docker eller publik drift.
4. Eventuella loggutdrag eller skärmbilder som hjälper analysen.

## Viktigt

- Försök inte utnyttja bristen mer än nödvändigt för att visa problemet.
- Publicera inte detaljer öppet innan felet är åtgärdat.
- Dela inte personuppgifter, hemligheter eller autentiseringsuppgifter i klartext.
